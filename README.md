# Simple Unix shell

A Unix-like shell made with C. Originally derived from [this tutorial](https://brennan.io/2015/01/16/write-a-shell-in-c/). Works on Linux.

### Current commands:

* cd
* ls
* mkdir
* touch
* rm
* chmod
* help
* echo
* clear
* tree
* pwd
* exit

### Compile and run with:

```
./build.sh
```

If running for the first time use:

```
chmod +x build.sh
```
